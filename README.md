# blogtakin

Astro blog configured for Cloudflare Pages with:

- `@astrojs/cloudflare` adapter
- `@astrojs/sitemap` sitemap generation
- Markdown posts under `src/pages/posts/`
- Author pages under `src/pages/authors/`
- Cloudflare Pages redirects in `public/_redirects`

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```
